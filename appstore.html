AppStore JSON APIs

This is just an iOS AppStore Clone Application using various JSON APIs coming directly from Apple's iTunes Search API.

  • Category: iOS Application

  • Github: https://github.com/hahpuc/Appstore-Clone

  • Technologies:
  • Swift 5

  • XCode 12.0.1

  • JSON APIs coming directly from Apple's iTunes Search API

  • CocoaPods 1.9.3

  • Fetch image from Url by SDWebImage

  • Design Pattern: MVC
